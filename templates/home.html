Pages

Register

The registration page adds a new user to the Users table.

If the locale provided is not yet in the Locales table, it is added here.

Finally, an entry is added to the UsersInLocales table.

The form must have a constraint to only allow for locales that don't currently have users to be added by new users.

Login

The login page uses a select statement to find the userID, which is used to find the user's name, locale, and upcoming and available walks and activities.

Quarpedia

This is the user's main page, where they can see their current activity.

To change any of this content, the user needs to navigate to other links:

  • locales in order to add a new place to walk to
  • walks to update locale
  • activities to add activities to do
